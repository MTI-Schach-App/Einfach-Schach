# einfach-schach

A chess app with focus on good UX for old people written in NextJS by students of the University of Applied Sciences Magdeburg

## How to use it?

Install:
```bash
git clone https://github.com/automaqqt/einfach-schach && cd einfach-schach
npm install
```

Use in development:
```bash
npm run dev
```

Use in production:
```bash
npm run build
npm run dev
```

Extras:
```bash
# Lint
npm run lint

# Prettier
npm run prettier
```
